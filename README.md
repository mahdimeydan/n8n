# Deploy n8n on Render

> [!IMPORTANT]
> **View full deployment instructions in the [**Render docs**](https://render.com/docs/deploy-n8n).**

This template defines a [`render.yaml`](https://github.com/render-examples/n8n/blob/main/render.yaml) file you can use to deploy [n8n](https://n8n.io/) on Render. Click **Use this template** in the upper right to copy this template into your account as a new repo.

The `render.yaml` file defines the following resources:

- A web service that pulls and runs the official n8n Docker image
- A Render Postgres database that stores n8n data

Each of the above uses a free instance type by default.

## 🚀 نصب و راه‌اندازی

### 📂 آپلود به GitHub

1. **کلون یا دانلود**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/quran-website.git
   cd quran-website
   ```

2. **راه‌اندازی Git (اگر از ابتدا شروع می‌کنید)**:
   ```bash
   chmod +x git_setup.sh
   ./git_setup.sh
   ```

3. **اتصال به GitHub**:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/quran-website.git
   git branch -M main
   git push -u origin main
   ```

### 🌐 آپلود به cPanel

1. **دانلود فایل‌ها**:
   - فایل‌ها را از GitHub دانلود کنید
   - یا مستقیماً ZIP فایل دانلود کنید

2. **آپلود روی هاست**:
   - وارد cPanel شوید
   - فایل‌ها را در پوشه `public_html` آپلود کنید
   - یا آن‌ها را در زیرپوشه‌ای مثل `quran` قرار دهید

3. **تست**:
   - سایت را در مرورگر باز کنید
   - از عملکرد صحیح اطمینان حاصل کنید

### 🔗 GitHub Pages (میزبانی رایگان)

1. **فعال‌سازی GitHub Pages**:
   - وارد تنظیمات repository شوید
   - بخش "Pages" را پیدا کنید
   - Source: "Deploy from a branch"
   - Branch: "main"

2. **آدرس سایت**:
   - `https://YOUR_USERNAME.github.io/quran-website`
