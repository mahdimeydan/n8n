# Deploy n8n on Render

> [!IMPORTANT]
> **View full deployment instructions in the [**Render docs**](https://render.com/docs/deploy-n8n).**

This template defines a [`render.yaml`](https://github.com/render-examples/n8n/blob/main/render.yaml) file you can use to deploy [n8n](https://n8n.io/) on Render. Click **Use this template** in the upper right to copy this template into your account as a new repo.

The `render.yaml` file defines the following resources:

- A web service that pulls and runs the official n8n Docker image
- A Render Postgres database that stores n8n data

Each of the above uses a free instance type by default.

# قرآن کریم - وب سایت جامع قرآنی

وب سایت جامع و حرفه‌ای قرآن کریم با امکانات کامل و رابط کاربری زیبا و مدرن

## ✨ امکانات

### 🕌 ویژگی‌های اصلی
- **نمایش کامل قرآن**: تمامی 114 سوره با آیات کامل
- **ترجمه فارسی**: چندین ترجمه معتبر فارسی (فولادوند، مکارم شیرازی، انصاریان)
- **ترجمه انگلیسی**: ترجمه‌های معتبر انگلیسی
- **پخش آنلاین**: تلاوت آیات با صدای قاریان مشهور
- **جستجوی پیشرفته**: جستجو در متن قرآن و ترجمه‌ها
- **علاقه‌مندی‌ها**: ذخیره آیات مورد علاقه
- **تنظیمات شخصی‌سازی**: تنظیم فونت، پوسته، و گزینه‌های نمایش

### 🎨 طراحی و رابط کاربری
- **طراحی مدرن**: رابط کاربری زیبا و حرفه‌ای
- **پوسته تیره/روشن**: امکان تغییر پوسته
- **ریسپانسیو**: سازگار با تمامی دستگاه‌ها
- **انیمیشن‌های نرم**: تجربه کاربری روان و زیبا
- **فونت‌های زیبا**: فونت‌های فارسی و عربی مناسب

### 📱 سازگاری
- **مرورگرهای مدرن**: Chrome, Firefox, Safari, Edge
- **دستگاه‌های موبایل**: iOS و Android
- **تبلت و دسکتاپ**: تمامی اندازه صفحه‌ها
- **cPanel**: آماده برای آپلود روی هاست‌های cPanel

## 🚀 نصب و راه‌اندازی

### روش ساده (برای cPanel)

1. **دانلود فایل‌ها**:
   - تمامی فایل‌های پروژه را دانلود کنید
   - فایل‌ها را در یک پوشه جمع‌آوری کنید

2. **آپلود روی هاست**:
   - وارد cPanel شوید
   - فایل‌ها را در پوشه `public_html` آپلود کنید
   - یا آن‌ها را در زیرپوشه‌ای مثل `quran` قرار دهید

3. **تست**:
   - سایت را در مرورگر باز کنید
   - از عملکرد صحیح اطمینان حاصل کنید

### ساختار فایل‌ها

```
├── index.html              # صفحه اصلی
├── assets/
│   ├── css/
│   │   ├── styles.css      # استایل‌های اصلی
│   │   ├── responsive.css  # استایل‌های ریسپانسیو
│   │   └── animations.css  # انیمیشن‌ها
│   └── js/
│       ├── config.js       # تنظیمات
│       ├── api.js          # API ها
│       └── main.js         # کد اصلی
└── README.md               # این فایل
```

## 🔧 تنظیمات API

این وب سایت از API های معتبر زیر استفاده می‌کند:

- **AlQuran.cloud**: برای متن قرآن و ترجمه‌ها
- **Fawaz Ahmed API**: برای ترجمه‌های اضافی
- **Audio APIs**: برای فایل‌های صوتی تلاوت

همه API ها رایگان و بدون نیاز به کلید API هستند.

## 📖 راهنمای استفاده

### 🏠 صفحه اصلی
- نمایش آمار کلی قرآن
- دکمه‌های دسترسی سریع
- امکان رفتن به آیه تصادفی

### 📚 مطالعه سوره‌ها
1. روی "سوره‌ها" کلیک کنید
2. سوره مورد نظر را انتخاب کنید
3. از فیلترهای "مکی" و "مدنی" استفاده کنید
4. روی "مطالعه" کلیک کنید

### 🔍 جستجو
1. وارد بخش "جستجو" شوید
2. کلمه یا عبارت مورد نظر را تایپ کنید
3. از فیلترها استفاده کنید
4. روی نتایج کلیک کنید

### ❤️ علاقه‌مندی‌ها
- روی آیکون قلب در هر آیه کلیک کنید
- آیات ذخیره شده در بخش "علاقه‌مندی‌ها" قابل مشاهده است

### ⚙️ تنظیمات
- اندازه فونت عربی و فارسی
- انتخاب قاری
- انتخاب ترجمه
- تنظیمات پوسته و نمایش

## 🔧 شخصی‌سازی

### تغییر رنگ‌ها:
فایل `assets/css/styles.css` را ویرایش کنید و متغیرهای CSS را تغییر دهید:

```css
:root {
    --primary-color: #2E7D32;  /* رنگ اصلی */
    --accent-color: #FF6B35;   /* رنگ تاکیدی */
}
```

### اضافه کردن ترجمه:
فایل `assets/js/config.js` را ویرایش کنید و ترجمه جدید اضافه کنید.

### تغییر فونت:
فونت‌های جدید را در فایل CSS اضافه کنید.
